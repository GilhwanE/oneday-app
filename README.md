2. CRUD 기능

   - read : 사용자는 자신의 기록을 조회 할 수 있습니다. (필터기능)
   - update : 사용자는 자신의 기록을 수정 할 수 있습니다.
   - delete : 사용자는 자신의 기록을 삭제 할 수 있습니다.
